A web-based application developed using Java, Servlets, JDBC, and HTML to manage warehouse shipment and dispatch operations.
The system helps track shipments, manage dispatch records, and streamline warehouse logistics.

PROJECT OVERVIEW:
The Warehouse Shipment Dispatch Management System is designed to automate and simplify shipment handling in warehouses.
It enables administrators to manage shipment entries, dispatch details, and monitor records efficiently.

FEATURES:
Add new shipment details
Dispatch shipment records
View shipment history
Manage warehouse operations
Database integration using JDBC
Simple and user-friendly interface

TECHNOLOGIES USED:
Java
Java Servlets
JDBC
HTML 
Oracle 
Eclipse IDE
